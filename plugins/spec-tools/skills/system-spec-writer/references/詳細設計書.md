## この仕様書の記載漏れチェック観点

- 全APIエンドポイントが記載されているか
- 全テーブル・全カラムが記載されているか
- APIはリクエスト/レスポンスの例があるか
- テーブル定義にカラムの型・制約が記載されているか
- CSVダウンロード機能がある場合、出力項目一覧・文字コード・出力範囲が記載されているか
